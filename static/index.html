Artwork Classifier

This is an artwork classifier, which determines the artist of a painting based on a deep neural network (DNN). The neural network has been trainined with paintings of approximately 200 well-known artists such as Claude Monet and Pablo Picasso.

Classification Result

The classifier is based on a deep neural network implemented and trained with Tensorflow. The model architecture is constituted by a multi-layer Convolutional Neural Network, which is followed by a fully connected layer and completed with a Softmax Readout Layer.

Training was done based on a Cross-Entropy error function.
